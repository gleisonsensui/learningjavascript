Data list

A tag datalis serve para carregar uma lista, dentro de um form, a partir de um input, servindo como um filtro autocomplete de seleção de dados.

Progress Section

A tag progress serve para criar barras de progresso ou carregamento sem o auxilio de CSS ou JavaScript, somente com a atualização dos atributos de valores de progresso. Trata-se de uma barra de loading nativa no html5.
